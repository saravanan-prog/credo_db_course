# ➕ What INSERT does

It lets you store new records in your database.

### 🧱 Basic syntax

##### 1. Insert with column names (recommended)
```
INSERT INTO employees (name, salary, department)
VALUES ('Saravanan', 50000, 'IT');
```
##### 2. Insert without specifying columns
```
INSERT INTO employees
VALUES ('Saravanan', 50000, 'IT');

⚠️ This works only if you provide values in the exact column order.
```
##### 3. Insert multiple rows
```
INSERT INTO employees (name, salary, department)
VALUES 
('Arun', 40000, 'HR'),
('Priya', 60000, 'Finance');

```

### 🔍 Key points
- Always match data types (e.g., numbers, strings).
- Strings should be in quotes ' '.
- Safer to mention column names to avoid mistakes.

### ⚠️ Common mistakes
- Missing values
- Wrong data type
- Not matching column order
