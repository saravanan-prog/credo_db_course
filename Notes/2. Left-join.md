###  LEFT JOIN

Returns:

- All rows from the left table
- Matching rows from the right table
- If no match → NULL


#### Example Tables : customers    -    orders

```
| customer_id | name  |           | order_id | customer_id |
| --------    | ----- |           | ---------| ------------|
| 1           | Ravi  |           | 101      | 1           |
| 2           | Priya |           | 102      | 2           |
| 3           | Arun  |           | 103      | 4           |

```

#### Query
```
SELECT customers.name, orders.order_id
FROM customers
LEFT JOIN orders
ON customers.customer_id = orders.customer_id;
```


#### Result
| name  | order_id |
| ----- | -------- |
| Ravi  | 101      |
| Priya | 102      |
| Arun  | NULL     |


#### Explanation:

- All customers are shown.
- Arun has no order, so NULL.
