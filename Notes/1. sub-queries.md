### 1. Subqueries

A subquery is a query written inside another query.

##### Syntax
```
SELECT column_name
FROM table_name
WHERE column_name operator (
    SELECT column_name
    FROM table_name
);
```
##### Example

Find employees whose salary is greater than average salary.
```
SELECT name, salary
FROM employees
WHERE salary > (
    SELECT AVG(salary)
    FROM employees
);
```
##### Explanation
- Inner query calculates average salary.
- Outer query finds employees earning more than that.
