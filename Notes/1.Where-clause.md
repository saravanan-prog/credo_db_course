## WHERE

In SQL, WHERE is used to filter data—it tells the database which rows to select, update, or delete.

### 🔍 What WHERE does

It applies a condition so only matching rows are affected.

🧱 Basic syntax
```
SELECT column_name
FROM table_name
WHERE condition;
```
### 📊 Common examples

#### 1. Use with SELECT
```
SELECT * FROM employees
WHERE salary > 50000;
```
➡️ Shows employees with salary greater than 50,000.

#### 2. Use with UPDATE

```       
UPDATE employees
SET salary = 60000
WHERE id = 1;
```
➡️ Updates only one specific row.

#### 3. Use with DELETE
```
DELETE FROM employees
WHERE department = 'HR';

➡️ Deletes only HR employees.

```

## 🔑 Common operators

| Operator  | Meaning               |
| --------- | --------------------- |
| `=`       | Equal                 |
| `>` `<`   | Greater / Less        |
| `>= <=`   | Greater/Less or equal |
| `!=`      | Not equal             |
| `AND`     | Both conditions true  |
| `OR`      | Any condition true    |
| `LIKE`    | Pattern matching      |
| `IN`      | Match multiple values |
| `BETWEEN` | Range                 |


### 🔹 = (Equal)
```
SELECT * FROM employees
WHERE department = 'IT';

➡️ Gets employees who work in IT
```
### 🔹 > and < (Greater / Less)
```
SELECT * FROM employees
WHERE salary > 50000;

➡️ Salary greater than 50,000

SELECT * FROM employees
WHERE salary < 30000;

```
### 🔹 >= and <= (Greater/Less or Equal)
```

SELECT * FROM employees
WHERE salary >= 50000;
SELECT * FROM employees
WHERE salary <= 30000;
```

#### 🔹 != (Not Equal)
```

SELECT * FROM employees
WHERE department != 'HR';

➡️ Excludes HR employees
```


#### 🔹 AND (Both conditions true)
```

SELECT * FROM employees
WHERE department = 'IT' AND salary > 40000;
```

#### 🔹 OR (Any condition true)
```

SELECT * FROM employees
WHERE department = 'IT' OR department = 'HR';
```

#### 🔹 LIKE (Pattern matching)
```

SELECT * FROM employees
WHERE name LIKE 'S%';

➡️ Names starting with "S"

SELECT * FROM employees
WHERE name LIKE '%an';

➡️ Names ending with "an"
```

#### 🔹 IN (Match multiple values)
```

SELECT * FROM employees
WHERE department IN ('IT', 'HR', 'Finance');

```

#### 🔹 BETWEEN (Range)
```

SELECT * FROM employees
WHERE salary BETWEEN 30000 AND 60000;


➡️ Salary between 30k and 60k
```
### 💡 Quick tip
```

AND     → narrows results
OR      → broadens results
LIKE    → flexible text searc
```
