# 🗑️ What DELETE does

It deletes specific rows or all rows, but keeps the table structure intact.

### 🧱 Basic syntax
```
DELETE FROM table_name
WHERE condition;
```
### 📊 Common examples

#### 1. Delete specific rows
```
DELETE FROM employees
WHERE name = 'Saravanan';

➡️ Removes only rows matching the condition.
```
#### 2. Delete multiple rows
```
DELETE FROM employees
WHERE salary < 30000;
```
#### 3. Delete all rows
```
DELETE FROM employees;

➡️ Clears all data, but the table still exists.
```
#### ⚠️ Important notes
- Always use WHERE to avoid deleting everything unintentionally.
- Can be rolled back if used inside a transaction (depends on DB).
- Slower than TRUNCATE for large tables.
