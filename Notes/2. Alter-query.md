# 🔧 What ALTER does

It lets you change the structure of a table without deleting it. For example:

- Add a new column
- Remove a column
- Modify a column’s data type
- Rename a column or table
🧱 Common examples

#### 1. Add a column
```
ALTER TABLE employees
ADD email VARCHAR(100);
```
#### 2. Modify a column
```
ALTER TABLE employees
MODIFY email VARCHAR(150);
```
#### 3. Drop (delete) a column
```
ALTER TABLE employees
DROP COLUMN email;
```
#### 4. Rename a column
```
ALTER TABLE employees
RENAME COLUMN email TO work_email;
```
