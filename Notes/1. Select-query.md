
# 💡 Simple idea - DML
```
SELECT = read data
INSERT = add data
UPDATE = modify data
DELETE = remove data
```



# 🔍 What SELECT does

It lets you query and view data from one or more tables.

### 🧱 Basic syntax
```
SELECT column1, column2
FROM table_name;
```
### 📊 Common examples

##### 1. Select all columns
```
SELECT * FROM employees;
```
➡️ Retrieves all data from the employees table.

##### 2. Select specific columns
```
SELECT name, salary FROM employees;

➡️ Shows only name and salary.
```
##### 3. Using a condition (WHERE)
```
SELECT * FROM employees
WHERE salary > 50000;

➡️ Filters rows based on a condition.
```
##### 4. Sorting results (ORDER BY)
```
SELECT * FROM employees
ORDER BY salary DESC;

➡️ Sorts data (highest salary first).
```
##### 5. Limiting results
```
SELECT * FROM employees
LIMIT 5;

➡️ Returns only 5 rows.
```



### 🧠 Key clauses used with SELECT
```
WHERE       → filter rows
ORDER BY    → sort data
GROUP BY    → group rows
HAVING      → filter groups
JOIN        → combine tables
```
