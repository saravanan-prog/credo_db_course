# Data Types?

Data types define what kind of data a column can store.

#### 🔢 1. Numeric Types

Used for numbers
```
INT            → whole numbers (Example: 1, 100, 5000)
FLOAT / DOUBLE → decimal numbers (Example: 10.5, 99.99)
```
#### 📝 2. String (Text) Types

Used for text
```
VARCHAR(n)   → variable-length text (Example: name, city)
CHAR(n)      → fixed-length text
TEXT         → large text (paragraphs)
```
#### 📅 3. Date & Time Types

Used for date and time
```
DATE     → YYYY-MM-DD (Example: 2026-04-29)
TIME     → HH:MM
DATETIME → date + time
```
#### ✔ 4. Boolean Type
```
BOOLEAN → true / false (stored as 1 or 0 in MySQL)
```

#### 🧠 Simple Table Example
```        
CREATE TABLE students (
  id INT,
  name VARCHAR(50),
  age INT,
  marks FLOAT,
  created_at DATETIME
);

```
#### 🎯 Easy Memory Trick

```
INT → numbers
VARCHAR → text
DATE → date
FLOAT → decimal
```
