### 📌 What is CREATE?
-  It comes under DDL (Data Definition Language)
- CREATE is used to create a database or a table.

#### 🧠 1. Create Database

```
CREATE DATABASE college;
```
✔ Creates a new database named college

#### 🧠 2. Use Database
```
USE college;
```
✔ Selects the database to work with

#### 🧠 3. Create Table
```
CREATE TABLE students (
  id INT PRIMARY KEY,
  name VARCHAR(50),
  age INT
);
```

- Creates a table named students
- Defines columns and their data types
