## 🔄 What UPDATE does

It lets you change values of one or more columns in existing rows.

### 🧱 Basic syntax
```
UPDATE table_name
SET column1 = value1, column2 = value2
WHERE condition;
```
### 📊 Common examples

##### 1. Update a single column
```
UPDATE employees
SET salary = 55000
WHERE name = 'Saravanan';

➡️ Updates salary for a specific employee.
```
##### 2. Update multiple columns
```
UPDATE employees
SET salary = 60000, department = 'IT'
WHERE id = 1;
```
##### 3. Update all rows (⚠️ risky)
```
UPDATE employees
SET salary = 50000;

➡️ Updates every row in the table.
```
##### ⚠️ Important notes
- Always use a WHERE clause unless you want to update all rows.
- Changes affect existing data permanently (unless using transactions).
- Double-check conditions before executing.
