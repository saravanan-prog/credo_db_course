#### 1. INNER JOIN

Returns only the matching rows from both tables.


#### Example Tables : customers    -    orders

```
| customer_id | name  |           | order_id | customer_id |
| --------    | ----- |           | ---------| ------------|
| 1           | Ravi  |           | 101      | 1           |
| 2           | Priya |           | 102      | 2           |
| 3           | Arun  |           | 103      | 4           |

```
  
### Query
```
SELECT customers.name, orders.order_id
FROM customers
INNER JOIN orders
ON customers.customer_id = orders.customer_id;

```
### Result

| name  | order_id |
| ----- | -------- |
| Ravi  | 101      |
| Priya | 102      |
