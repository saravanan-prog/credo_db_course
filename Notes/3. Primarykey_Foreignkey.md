## Primary Key

👉 A Primary Key is a column that:

- Uniquely identifies each row
- Cannot be duplicate
- Cannot be NULL

#### Example

```
  CREATE TABLE students (
    id INT PRIMARY KEY,
    name VARCHAR(50)
 );
```

| id | name |
| -- | ---- |
| 1  | Prem |
| 2  | Ravi |


## Foreign Key

👉 A Foreign Key is a column that:

- Connects one table to another
- Refers to a Primary Key in another table

```
CREATE TABLE courses (
  course_id INT,
  student_id INT,
  FOREIGN KEY (student_id) REFERENCES students(id)
);
```

| course_id | student_id |
| --------- | ---------- |
| 101       | 1          |



## Difference :  Primary Key vs Foreign Key

| Primary Key   | Foreign Key      |
| ------------- | ---------------- |
| Unique ID     | Reference ID     |
| In main table | In another table |
| No duplicates | Can repeat       |
